# Napoleon IT iOS App Dev Online course

### Урок 4. Продвинутый UI с использованием анимации и навигации между экранами

Реализовать TabBar с 3 вкладками:
1. На первой вкладке должна быть кнопка, открывающая второй экран через push (с помощью сегвея). На открывшемся экране должна быть кнопка, программно (не через сегвей, а через код) открывающая еще один экран (хоть модально, хоть добавляя в навигационный стэк). Со второго экрана на третий должен передаваться какой-либо текст. Этот текст на третьем экране должен появляться анимированной (анимация не важна).
2. На второй вкладке должна быть View и кнопка. По нажатию на кнопку должна запускаться цепочка анимаций (минимум 3 штуки), в конце которой View возвращается в исходную точку.
3. На третьей вкладке должно быть 3 View. По нажатию на первую с ней должна выполняться какая-либо анимация с использованием UIView.transition(from:..., to:...) (в качестве параметра to надо использовать одну из добавленных View). Вторая, по нажатию на нее, должна анимированно менять цвет и какой-нибудь констрэинт относительно края экрана (можно рандомно менять оба параметра, чтобы можно было нажимать несколько раз).
