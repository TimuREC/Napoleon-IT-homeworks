### Урок 6. Процесс дебаггинга в рамках Xcode SDK. Параллельный и асинхронный код для нетривиальных задач

1. На экране должна быть коллекция с тремя столбцами и несколькими строками. Над таблицей внизу экрана должны быть закреплена кнопка, по нажатию на которую начинается загрузка фото в ячейки с кэшированием. По нажатию на фотографию открывается новый экран с этой же фотографией (она должна быть одна). 
2. Получить любой json и распарсить его с помощью Codable.
