### Урок 8. Основы рефакторинга и правильного проектирования кода. Продвинутые инструменты в Swift

Задание 1.
В коллекцию с фотографиями из прошлых заданий добавить строку ввода (над коллекцией). При открывании клавиатуры поднимать таблицу так, чтобы при скролле до конца коллекции с открытой клавиатурой последний элементы было видно (то есть они должны быть прямо над клавиатурой. Высчитывание размеров клавиатуры и событие ее поднятия/закрытия делать через протокол (если метод протокола реализуется в его Extension и не переопределяется в классе, то метод убирать из тела протокола и оставлять только внутри его Extension).
