# Napoleon IT iOS App Dev Online course

### Урок 1. Основы языка Swift, абстрагированные от мобильной разработки

Задание 1.
Создать новый проект и новый репозиторий на гитхабе под него. Сменить цвет заднего фона LaunchScreen и добавить на экран (так, чтобы нижняя часть текста была ровно посередине) ваше ФИО. В сториборде добавить на View Controller, который является входной точкой, любой текст размера от 20 и любого цвета, кроме черного, на середину верхней половины экрана (чтобы на устройствах разного размера он был на одном и том же месте) и кнопку на середину нижней половины экрана. В ответ прикрепить ссылку на репозиторий.

> Например, можно создать класс "Фигура", наследником которого будут фигуры (треугольник, круг, прямоугольник). У этого класса будут read-only переменные name и cornerRadius, которые надо будет переопределить в наследниках. Также каждый наследник должен иметь метод расчета площади. Протокол будет создан для четырехугольных фигур, стороны которых попарно равны (прямоугольник, квадрат, параллелограм, параллелограмм). В нем будут переменные "ширина", "длина" и "Периметр" (переменная периметр должна иметь стандартную реализацию).
