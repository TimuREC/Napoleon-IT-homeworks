# Napoleon IT iOS App Dev Online course

### Урок 2. Базовая навигация по Xcode. Основные элементы SDK. Базовая верстка без кода 

Задание 1.
Создать новый проект и новый репозиторий на гитхабе под него. Сменить цвет заднего фона LaunchScreen и добавить на экран (так, чтобы нижняя часть текста была ровно посередине) ваше ФИО. В сториборде добавить на View Controller, который является входной точкой, любой текст размера от 20 и любого цвета, кроме черного, на середину верхней половины экрана (чтобы на устройствах разного размера он был на одном и том же месте) и кнопку на середину нижней половины экрана. В ответ прикрепить ссылку на репозиторий.

Задание 2.
Перейти в отдельную ветку. Создать View в XIB-файле с любыми несколькими элементами, например, текст слева и свитч справа от него. Для такого примера у текста должно быть два Constraint'а справа, один с более высоким приоритетом до свитча, второй с менее высоким до права края View (если вы придумываете свои элементы и их расположение, то такие два констрэинта также должны быть между ними). Создать новый View Controller, сделать его входной точкой в приложение и накидать такие же два элемента внутрь StackView в треть экрана. Верхние две трети экрана занять любой картинкой. Сделать Merge Request этой ветки в основную. В ответ прикрепить ссылку на Merge Request.
