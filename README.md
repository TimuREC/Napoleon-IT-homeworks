# Napoleon IT iOS App Dev Online course

### Урок 2. Базовая навигация по Xcode. Основные элементы SDK. Базовая верстка без кода 

Задание 1.
Создать новый проект и новый репозиторий на гитхабе под него. Сменить цвет заднего фона LaunchScreen и добавить на экран (так, чтобы нижняя часть текста была ровно посередине) ваше ФИО. В сториборде добавить на View Controller, который является входной точкой, любой текст размера от 20 и любого цвета, кроме черного, на середину верхней половины экрана (чтобы на устройствах разного размера он был на одном и том же месте) и кнопку на середину нижней половины экрана. В ответ прикрепить ссылку на репозиторий.
